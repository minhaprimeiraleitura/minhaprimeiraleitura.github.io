Livro Infantil para Alfabetização

Conheça o livro infantil para ajudar a criança na alfabetização. Este livrinho é formado apenas por historinhas simples, com palavras com no máximo 2 sílabas e fáceis de serem pronunciadas. Tudo o que a criança precisa para começar a ser alfabetizada.

Como é o Passatempo da coleção?

Para cada livro de leitura estamos lançando um passatempo utilizando as mesmas características do respectivo livro.

É ideal para a criança também treinar e fazer atividades específicas para melhorar a leitura, a escrita, o foco, a concentração e a criatividade!

Aproveite para levar os dois ao mesmo tempo, assim você economiza também.

Os próximos passatempos serão lançados logo após cada livro de leitura for lançado.
